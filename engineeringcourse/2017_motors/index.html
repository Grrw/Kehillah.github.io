My Project

By Avi Ben Tov

The purpose of this contraption is to show off how the the keypad is used to interact with the motor array. It is part of a vending machine that I am constructing

The materials used in this project consist of:

  • An arduino board (any model will suffice)
  • Some wires for cabling
  • A plexiglass base
  • Two Stepper Motors
  • Two Motor Controlers (One for each motor)
  • Two 9 volt (One for each motor controler)
  • A small micro usb cable to plug in to the computer
  • One Keypad
